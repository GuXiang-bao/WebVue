# WebVue
留言墙作业 vue+nodejs+mysql

**本代码库/文件/项目仅供学习、研究之用，不得用于商业目的或任何其他非法用途。**

本人使用开发环境：
"vue": "^3.2.13",
"nodejs":"v18.12.1",
"mysql":"8.0.36",

**代码在[master](https://github.com/GuXiang-bao/WebVue/edit/master)分支**

# 运行方法
运行方法在web和serve输入命令
```
npm install //下载依赖包 
```
serve为后端运行命令
```
node index.js
```
web为前端运行命令
```
npm run serve
```
